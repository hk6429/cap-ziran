# 自然科練功房

國中教育會考與國中基測自然科歷屆試題練習網站。可依年度、場次、內容分類、難度與鑑別度選題，並提供整回模考、教師出卷、錯題複習、學習歷程及逐題解析。

## 本機預覽

```sh
npx serve .
```

開啟終端機顯示的本機網址即可使用。本站是純靜態網站，不需建置步驟。

## 測試

```sh
node --test "test/**/*.test.mjs"
```

## 資料原則

- 題目與答案以正式歷屆試題及參考答案為準。
- 選項比例、通過率與鑑別度只匯入來源明載的數值，不自行推算。
- 四選項比例未納入未作答與複選，因此合計可能略低於 100%。
- `data/answer-distributions.js` 由正式分析資料產生；可用 `scripts/import-answer-distributions.mjs` 重新匯入已核對的 JSON。

## 部署

專案可部署至 Vercel、Cloudflare Pages 或 Netlify；發布目錄為專案根目錄。
